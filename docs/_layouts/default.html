Description

Experimenting with different dimensions of Convolutional Neural Network’s Filters in order to find the most accurate solution for Traffic Signs Classification.


How to use

  • The easiest way is to clone code on Kaggle here.
  • Another option would be to download Jupyter Notebook file and use it at local machine. But in this case, you need to download also dataset that is located on Kaggle here. Also, change the location of dataset in the second cell, pointing to the downloaded file, e.g.: 
    # Opening file for reading in binary mode
with open('../input/traffic-signs-preprocessed/data2.pickle', 'rb') as f:
  data = pickle.load(f, encoding='latin1')  # dictionary type
